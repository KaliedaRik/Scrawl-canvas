Test purpose

  • Create some elements to be snippet-ized (warning: made-up word!) in the HTML file
  • Find and snippet-ize the elements (according to their class values)
  • Check that the canvases added to each Scrawl-canvas enhanced snippet size correctly to their element and do not block user interaction with the element
  • Check that snippets interact with mouse cursor movements and clicks over the element/canvas
  • Check that snippet canvases correctly mimic their elements, in particular to width/height updates
  • Check that snippet animations disable themselves when they move out of the viewport, and restart when they come back into view
  • Snippet-ize the 'reportmessage' <p> element (with no canvas added to it); give it functionality to report back on page performance eg frames-per-second etc

Note that the main file loads the minified version of the library, while the snippets load the source version - test to see if any conflicts from the two arise.
